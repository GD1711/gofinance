# SaldoFuturo

App financeiro previsivo com arquitetura limpa e design premium.

## Estrutura do Projeto

```
src/
├── app/              # Rotas Next.js (App Router)
├── domain/           # Regras de negócio (entidades, serviços)
├── application/      # Casos de uso
├── infrastructure/   # APIs, repositórios, banco
├── ui/              # Componentes, layouts, design system
├── hooks/           # Custom hooks React
├── utils/           # Funções auxiliares
└── types/           # Tipos TypeScript compartilhados
```

## Tecnologias

- **Frontend**: Next.js 14, React 18, TypeScript
- **Styling**: Tailwind CSS
- **Animações**: Framer Motion
- **Ícones**: Lucide React
- **Gráficos**: Recharts

## Começar

```bash
npm install
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000) no navegador.

## Princípios

- Arquitetura limpa
- Código legível > código "esperto"
- Mobile first
- UX humana e clara
